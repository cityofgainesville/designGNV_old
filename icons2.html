Icons


To use icons as images for presentations:

Click on the icon you wish to download. It will open in a new page, then right click and save the file in your desired location.


To use icons in HTML for websites:

1. Insert the Font Awesome script into the head of your code.

<!-- Start FontAwesome Kit -->
<script src="https://kit.fontawesome.com/7c02223738.js" crossorigin="anonymous"></script>
<!-- End FontAwesome Kit. -->

2. Copy the HTML code below the icon you wish to use and insert it into your code.
